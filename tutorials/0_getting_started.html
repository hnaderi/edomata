Getting started

Install

libraryDependencies += "io.github.hnaderi" %% "edomata-core" % "0.0-ce14ebf-SNAPSHOT"

or for scala.js

libraryDependencies += "io.github.hnaderi" %%% "edomata-core" % "0.0-ce14ebf-SNAPSHOT"

Imports

import edomata.core.*
import edomata.syntax.all.* // for convenient extension methods

Decision

It all start with decisions!
Decision models programs that decide in a event driven context, these programs are pure and can:
- accept events
- reject with errors
- stay indecisive!

val d1 = Decision(1)
// d1: Decision[Nothing, Nothing, Int] = InDecisive(1)
val d2 = Decision.accept("Missile Launched!")
// d2: Decision[Nothing, String, Unit] = Accepted(
//   Singleton("Missile Launched!"),
//   ()
// )
val d3 = Decision.reject("No remained missiles to launch!")
// d3: Decision[String, Nothing, Nothing] = Rejected(
//   Singleton("No remained missiles to launch!")
// )

decisions are composable

val d4 = d1.map(_ * 2)
// d4: Decision[Nothing, Nothing, Int] = InDecisive(2)
val d5 = d2 >> d1
// d5: Decision[Nothing, String, Int] = Accepted(
//   Singleton("Missile Launched!"),
//   1
// )
val d6 = d5 >> d3
// d6: Decision[String, String, Nothing] = Rejected(
//   Singleton("No remained missiles to launch!")
// )

you can also use for comprehensions:

val d7 = for {
  i <- Decision.pure(1)
  _ <- Decision.accept("A") // accepting one event
  _ <- Decision.accept("B", "C") // accepting several events
  j <- Decision.acceptReturn(i * 2)("D", "E") // accepting several events and returning a value
} yield i + j
// d7: Decision[Nothing, String, Int] = Accepted(
//   Append(
//     Singleton("A"),
//     Append(
//       Append(Singleton("B"), Singleton("C")),
//       Append(Singleton("D"), Singleton("E"))
//     )
//   ),
//   3
// )

or in presence of cats syntax:

import cats.implicits.*     // for cats related stuff     // for cats related stuff

val d8 = (d1, d7).mapN(_ + _)
// d8: Decision[Nothing, String, Int] = Accepted(
//   Append(
//     Singleton("A"),
//     Append(
//       Append(Singleton("B"), Singleton("C")),
//       Append(Singleton("D"), Singleton("E"))
//     )
//   ),
//   4
// )
val d9 = List.range(1, 3).traverse(Decision.accept(_))
// d9: Decision[Nothing, Int, List[Unit]] = Accepted(
//   Append(Singleton(1), Singleton(2)),
//   List((), ())
// )
val d10 = Decision(List.range(1, 3)).sequence[List, Int]
// d10: List[Decision[Nothing, Nothing, Int]] = List(
//   InDecisive(1),
//   InDecisive(2)
// )
